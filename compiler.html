static
The term "static" can be used for "bodies".If a body is static, then it remains non-moveable forever.
def
"def" syntax abbreviation stands for "default". So whenever you set or change a value with def. The previous value will be set to the default value
boolean
the "boolean" expression is a logical statement that includes the values "true" or "false".
eventListener
The eventListener interface represents an object that can handle an event dispatched by an EventTarget object.
EventTarget
EventTarget is a DOM interface implemented by objects that can receive events and may have listeners for them.
DOM
"DOM", which is the abbreviation for "Document Object Model", connects web pages to scripts or programming languages by representing the structure of a document.
key
The key property returns the value of the key pressed by the user.
code
A code is the text written in the compiler which is gonna be executed by UYC Protocols.
iteration
An iteration is the repetition of a process that will end after a moment.
timeout
An timeout is a cancellation or cessation that automatically occurs when a predefined interval of time has passed.
UGI
The UGI, which is the abbreviation of "User Graphic Interface" is a user interface based on a display system that uses graphical elements.
Velocity
The Velocity of an object is the rate of change of its position with respect to a frame of reference, and is a function of time. Velocity is equivalent to a specification of an object's speed and direction of motion (e.g. 60 km/h to the x). Velocity is not the same as force.
Friction
The Friction is the force that resists the sliding or rolling of one solid body over another.
Velocity properties
The Velocity properties allow or deny the force of -x,+x,-y and +y to the body. An example: {up:true,down:true,left:true,right:true}
Heading
The Headings are titles or subtitles that you want to display. Headings are defined from a range between 1 and 6.
src
The src attribute specifies the URL of an external file. The file can be a script,audio,image...type.
Point light
The Point light is a light that gets emitted from a single point in all directions. A common use case for this is to replicate the light emitted from a bare lightbulb.
Particle function
The Particle function is the property that can be added as a function for the particles. Example: forceX=value or forceY=value
Mass
The Mass of an object (body) mass determines the strength of its gravitational attraction to other objects.
Inertia
The Inertia is the resistance of any object (body) to any change in its velocity. This includes changes to the object's speed, or direction of motion.
Camera
The Camera system aims at controlling the 2D camera to display the view.
